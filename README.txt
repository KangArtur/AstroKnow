Тема: Астрономический форум AstroKnow, реализованный на языке программирования Python

Данный проект представляет собой сайт, где астрономы могут опубликовывать исследования, а пользователи - задавать вопросы.

Проект реализован на основе библиотеки flask

При регистрации пользователь выбирает должность: астроном или гость.

Пользователи-астрономы могут опубликовывать исследования.
Исследование может быть отредактировано или удалено только создателем.
К исследованию любой зарегистрированный пользователь может оставить комментарий

Пользователи-гости могут задавать вопросы.
Вопрос может быть отредактирован или удален только создателем.
К вопросу любой зарегистрированный пользователь может оставить ответ.

База данных, в которой хранится вся информация о пользователях, исследованиях, комментариях, вопросах и ответах создана с помощью SQLalchemy

Формы для регистрации, создания исследования, комментария, вопроса или ответа реализованы с помощью flask_wtf
Хеширование паролей реализовано с помощью werkzeug.security

AstroKnow позволяет пользователю увидеть некоторые астрономические изображения. Это было реализовано с помощью подключения к общедоступному NASA API

Список использованных API:
- APOD: Ежедневная красивая фотография космического тела или явления
- Mars Rover Photos: Фотографии, полученные с марсохода
- ASA Image and  Video Library: Поиск космических картинок по запросу

База данных хранится в файле data.db

Данные для таблиц базы данных хранятся в папке data

Форма для регистрации пользователя хранится в папке forms

Логотип сайта хранится в папке static

Шаблоны хранятся в папке emplates
